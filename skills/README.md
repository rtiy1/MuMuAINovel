# Skills 集成说明

本目录已集成第三方写作技能包：

- 来源仓库：`https://github.com/OUBIGFA/De-AI-Prompt-Enhancer-Writer-Booster-SKILL`

- `skills/good-writing/SKILL.md`
- `skills/de-AI-writing/SKILL.md`

## 目录结构

- `skills/good-writing/`
- `skills/good-writing/references/`
- `skills/de-AI-writing/`
- `skills/de-AI-writing/references/`
- `skills/de-AI-writing/tools/`
- `skills/OUBIGFA-skill-pack.meta.json`

## 使用建议

- 需要“增强文风质感”时，优先使用 `good-writing`。
- 需要“去 AI 味、保真改写”时，优先使用 `de-AI-writing`。
- 两者都依赖 `references` 下的规则文件，建议保持目录结构不变。
